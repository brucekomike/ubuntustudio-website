# Artwork related

## default ubiquity background

This is the background shown when choosing install from the live DVD boot menu, as opposed to choosing the default option. The wallpaper is change in the source for ubiquity.

**browse code:** [http://bazaar.launchpad.net/~ubuntu-installer/ubiquity/trunk/view/head:/bin/ubiquity-dm](http://bazaar.launchpad.net/~ubuntu-installer/ubiquity/trunk/view/head:/bin/ubiquity-dm) (at line 308 at the time of this writing)

**get the code:** bzr branch lp:ubiquity

**push code:** bzr push lp:~<username>/ubiquity/<description-of-change>
