## Ubuntu Studio-Flat

I'll be converting this set to the Tango naming spec and colorizing where appropriate.

[G-Flat SVG](http://art.gnome.org/themes/icon/1281) based on [FlatSVG](http://www.kde-look.org/content/show.php/Flat?content=12132&PHPSESSID=4621a2f6a2064dd037ffea6163cf0350) is a set I'm thinking of using as a base for our set.

![ICON-GFlatSVG-color.png](https://raw.githubusercontent.com/UbuntuStudio-Official/ubuntustudio-website/main/wiki/content/_assets/wiki_ubuntu_com/UbuntuStudio/Artwork/DIYConcept/Icons/ICON-GFlatSVG-color.png) Here's a couple of changes.

```
bzr branch http://bazaar.launchpad.net/~ubuntustudio-dev/ubuntustudio-icon-theme/hardy-artwork
```
