I built this icon set from other artists' work (twisted and mangled some, left others alone - don't worry it's all GPL) and built it upon the Hardy Human icon set. It's not really designed as a DIY theme, but the vector graphics somewhat suit this idea/feel. It's nowhere near finished, and still contains many of the gnome default icons that I've yet to cover over, but I'd like to get some feedback on this.
