## New Branding

In March 2022 Ubuntu Studio announced a [logo refresh](https://ubuntustudio.org/2022/03/a-new-logo-for-ubuntu-studio/), with the changes being incorporated on the 22.04 LTS Beta image.

### Clone from Git

Clone the current logos using:

git clone [https://code.launchpad.net/~ubuntustudio-art/ubuntustudio-artwork/+git/current-standard](https://code.launchpad.net/~ubuntustudio-art/ubuntustudio-artwork/+git/current-standard)

Clone past and current artwork using:

git clone [https://code.launchpad.net/~ubuntustudio-art/ubuntustudio-artwork](https://code.launchpad.net/~ubuntustudio-art/ubuntustudio-artwork)
