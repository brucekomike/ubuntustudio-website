# What is Ubuntu Studio?

## An Official Flavor of Ubuntu

Being an official [flavor](https://wiki.ubuntu.com/DerivativeTeam/Derivatives) of Ubuntu means the following:

- Ubuntu Studio shares [repositories](https://help.ubuntu.com/community/Repositories/Ubuntu) with Ubuntu, as does other official Ubuntu flavors. This means you can install the same packages from any official Ubuntu flavor. The main difference between flavors is which packages and settings get pre-installed when installing using one of the [installer ISOs](http://cdimage.ubuntu.com/).
- Most of the multimedia packages provided by Ubuntu Studio are directly imported from [Debian](http://debian.org) repositories, which means we do not package them. Read more about Debian/Ubuntu here: [http://wiki.ubuntu.com/Debian](http://wiki.ubuntu.com/Debian)

## Ubuntu Studio and Free Software Licenses

[FLOSS](http://en.wikipedia.org/wiki/Free_and_open_source_software) stands for Free Libre Open Source Software. With some minor exceptions, such as non-free wifi drivers in the kernel, all the applications you get with Ubuntu Studio are truly FLOSS.

## Volunteer Project

Ubuntu Studio is a community project, developed by volunteers. We develop Ubuntu Studio on our free time, and our only goal is to make Ubuntu Studio a great, free OS for linux multimedia.
