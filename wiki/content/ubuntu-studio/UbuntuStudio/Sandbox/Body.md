### Help with Development

We would LOVE to have you help us with development.

Not a "developer"? That's cool, because we still need you!

Contributors are welcome also. See our [HelpWithDevelopment](https://wiki.ubuntu.com/HelpWithDevelopment) page for how you can help!

**Introduction to Ubuntu Studio** Want to know what Ubuntu Studio is? Don't know JACK? Can't tell the -lowlatency kernel from a -generic kernel with uname?

Learn more about FOSS and Ubuntu Studio by clicking on our [LearnUbuntustudio](https://wiki.ubuntu.com/LearnUbuntustudio) link.

**Hardware** Need to know about hardware? Want to know what is recommended? Or the minimum requirements? Confused about the differences between USB, PCI, and firewire audio interfaces?

Then the [HardwareLink](https://wiki.ubuntu.com/HardwareLink) is where you want to go.

**Installation** Want to install Ubuntu Studio? We can help.

Learn to install Ubuntu Studio from DVD for a full install or "upgrade" from a vanilla Ubuntu install [HereLink](https://wiki.ubuntu.com/HereLink).

**Optimization** The default installation not enough for you? Want to squeeze more out of your system?

Then check out our [OptimizationLink](https://wiki.ubuntu.com/OptimizationLink) page!

**Using Ubuntu Studio** Want to learn how to use the applications in Ubuntu Studio? Then this is the link for you.

Follow our work flows or spend time with a detailed [YouTube](https://wiki.ubuntu.com/YouTube) tutorial video on our [UsingUbuntuStudio](https://wiki.ubuntu.com/UsingUbuntuStudio) page.

**Support** Confused or need help?

Click [HereLink](https://wiki.ubuntu.com/HereLink) to see information about our IRC, mailing list, and carrier pigeon support channels.

**Trouble Shooting** Like to figure stuff out yourself? We do too!

See our [TroubleShooting](https://wiki.ubuntu.com/TroubleShooting) page to figure it out!
