# Welcome to the Ubuntu Studio developer wiki!

Ubuntu Studio is an official flavor of [Ubuntu](https://ubuntu.com/) focused on creative production workflows.

Ubuntu Studio is a community effort, targeted towards all skill levels, from beginner to pro, and aims to be easy to install and easy to use, as well as providing all the tools necessary for any type of media content creation.

## What we do..

Our goal is to provide a polished [Debian](https://debian.org/) and Ubuntu based system for creative [workflows](https://ubuntustudio.org/tour/), while contributing fixes upstream and coordinating with users and developers through testing and feedback.

We aim at being the gateway for new users coming into the Linux world, wanting to use Linux for multimedia content creation, by providing applications and documentation, as well as being a portal to other GNU/Linux and FLOSS communities in the world.

# Please Join Us!

Interested in contributing? Go ahead, and join the Ubuntu Studio team! We are always looking for new contributors of any skill level. We are all volunteers.

You may find us at:

- [Ubuntu Studio on Discourse](https://discourse.ubuntu.com/c/flavors/ubuntustudio/188) - release notes and team discussions
- [Ubuntu Discourse Support and Help](https://discourse.ubuntu.com/c/support-and-help/306) - preferred support method
- [Matrix support chat](https://matrix.to/#/#ubuntustudio:ubuntu.com) - real-time support chat
- [ubuntustudio.org](https://ubuntustudio.org/) - official website
