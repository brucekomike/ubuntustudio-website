# Installation
## find the boot option of your computer
The fastest method to find the key combination of boot option is to refer the offical manual.

Here are some common key map for boot option
- `F12` `F11` `F7` `F8`
- some hardware can activate custom boot option from hardware settings
  - `F2` `Delete` `F12`

## use ubuntu studio installer
- refer to the gui installer
- not to connect network for time saving
  - the install extend media option can be done later as this command:
  - `sudo apt install ubuntu-restricted-extras`
  - packages for additional driver can be queried using this command:
  - `sudo ubuntu-drivers`
